Single customer quote card — initials avatar (never a stock photo), name/role/company.

```jsx
<Testimonial
  quote="Отправляем чертежи вечером — утром уже виден перевод с сохранённой разметкой таблиц."
  name="Роман Соколов"
  role="Руководитель проектов"
  company="промышленный холдинг"
  sample
/>
```

Set `sample` while the quote is illustrative/unsourced — drop it once a real, permissioned customer quote replaces the placeholder.
